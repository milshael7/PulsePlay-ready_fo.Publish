# PulsePlay

PulsePlay is a modern web platform for video sharing, live streaming, screen recording, and product showcasing. Users can watch live content, upload videos, and even sell products directly through the platform.  

## Key Features

- Live video streaming and on-demand video playback  
- Screen recording for tutorials or live presentations  
- Product showcasing and sales capabilities  
- User-friendly interface for browsing, searching, and interacting with content  

## Project Structure

- `frontend/` — React frontend with components, layouts, pages, styles, and routes  
- `backend/` — Node.js backend handling API routes, data services, and configurations  

PulsePlay combines video entertainment and e-commerce in a clean, modular architecture.
